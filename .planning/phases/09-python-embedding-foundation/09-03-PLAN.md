    Método start() async:
    - Guardia: guard state == .idle || state == .failed else { return }
    - Establecer state = .starting
    - Llamar findFreePort(); si devuelve 0, establecer state = .failed y return

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Swift app → subprocess Python | La app lanza un proceso hijo con executableURL verificado dentro del bundle; el proceso no tiene acceso de red externo — bound a 127.0.0.1 |
| Subprocess Python → loopback HTTP | El servidor uvicorn escucha exclusivamente en 127.0.0.1; no expuesto a la red local ni a internet |
| PID file /tmp/ → filesystem | El fichero /tmp/md-translator-python.pid es accesible por cualquier proceso del usuario; la mitigación está en la lectura defensiva (verificar que el proceso con ese PID realmente es python3 antes de SIGKILL) |
