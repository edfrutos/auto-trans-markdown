    Variables de descarga (exactas, no modificar):
    - PBS_RELEASE="20260510"
    - PBS_VERSION="3.11.15"
    - PBS_ARCH="aarch64-apple-darwin"
    - PBS_FLAVOR="install_only_stripped"
    - BUNDLE_DIR="$REPO_ROOT/python-bundle"
    - PBS_ARTIFACT="cpython-${PBS_VERSION}+${PBS_RELEASE}-${PBS_ARCH}-${PBS_FLAVOR}.tar.gz"
    - PBS_URL construida con las variables anteriores apuntando a github.com/astral-sh/python-build-standalone/releases/download/${PBS_RELEASE}/${PBS_ARTIFACT}

    Paso 1 — Descarga: curl -L --progress-bar "$PBS_URL" -o /tmp/cpython-standalone.tar.gz

    Paso 2 — Verificación de estructura del tarball (mitigación Pitfall 1 + A1):
    Usar tar tzf /tmp/cpython-standalone.tar.gz 2>/dev/null | head -1 para leer el primer entry.
    Si el primer entry NO es "python/", hacer echo de error descriptivo a stderr y exit 1 con el valor encontrado.

    Paso 3 — Extracción: rm -rf "$BUNDLE_DIR", mkdir -p "$BUNDLE_DIR", tar xzf con -C "$BUNDLE_DIR" --strip-components=1.

    Paso 4 — Export de requirements: uv export --format requirements-txt --no-dev --no-editable --no-emit-project --no-hashes > /tmp/bundle-requirements.txt (ejecutado desde REPO_ROOT para que uv.lock sea encontrado).

    Paso 5 — Instalación de dependencias: uv pip install -r /tmp/bundle-requirements.txt --target "$BUNDLE_DIR/lib/python${PBS_VERSION}/site-packages/" --python "$BUNDLE_DIR/bin/python3"

    Paso 6 — Smoke test (BUNDLE-02): "$BUNDLE_DIR/bin/python3" -c "import fastapi; print('OK — fastapi', fastapi.__version__)". Si falla: echo de error descriptivo en español a stderr y exit 1. Si pasa: mensaje de confirmación en stdout.

    Todos los mensajes de progreso en stdout con prefijo "→". Mensajes de error a stderr. El script debe ser ejecutable (aplicar chmod +x).

    NO añadir fallback con python3 -m pip (usar solo uv pip install según D-03). NO incluir WeasyPrint ni extras [pdf] ni [test]. NO hardcodear versiones de paquetes individuales — todo viene de uv.lock via uv export.
