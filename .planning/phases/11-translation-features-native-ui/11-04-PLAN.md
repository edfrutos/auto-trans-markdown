# Plan 11-04 — Carpeta de salida personalizada y apertura en Finder
## Phase: 11 — Translation Features & Native UI

**Estado:** PLANIFICADO  
**Objetivo:** Permitir al usuario elegir la carpeta donde se guardan los archivos traducidos (en lugar del `output/` temporal del servidor) y abrir el resultado directamente en Finder tras la traducción.

---

## Contexto

- El backend guarda traducciones de archivo en `output/` (ruta relativa al cwd del subprocess = `Resources/backend/`). El `OUTPUT_TTL_HOURS` las borra tras N horas.
- El usuario no tiene acceso fácil a esa ruta dentro del bundle — está en DerivedData o en `/Applications`.
- La solución más limpia para macOS: Security-Scoped Bookmarks para persistir acceso a la carpeta elegida entre sesiones, más un endpoint de descarga para obtener el `.md` traducido.
- **Alternativa más simple (v3.0):** tras la traducción, descargar el archivo desde `/api/translate/file` y guardarlo en la carpeta elegida vía Swift + `NSSavePanel` o folder bookmark.

---

## Tareas

### 1. `OutputManager.swift` — gestión de carpeta de salida

```swift
// OutputManager.swift
import Foundation
import AppKit

@MainActor
final class OutputManager {
    static let shared = OutputManager()

    private let bookmarkKey = "outputFolderBookmark"

    /// URL de la carpeta de salida guardada. Nil si el usuario no ha elegido ninguna.
    var outputFolderURL: URL? {
        guard let data = UserDefaults.standard.data(forKey: bookmarkKey) else { return nil }
        var stale = false
        return try? URL(
            resolvingBookmarkData: data,
            options: .withSecurityScope,
            relativeTo: nil,
            bookmarkDataIsStale: &stale
        )
    }

    /// Abre un NSOpenPanel para que el usuario elija la carpeta de salida y guarda el bookmark.
    func chooseFolderWithPanel() {
        let panel = NSOpenPanel()
        panel.title = "Carpeta para archivos traducidos"
        panel.canChooseFiles = false
        panel.canChooseDirectories = true
        panel.canCreateDirectories = true
        panel.prompt = "Elegir carpeta"
        guard panel.runModal() == .OK, let url = panel.url else { return }
        saveBookmark(for: url)
    }

    /// Guarda el archivo traducido en la carpeta de salida. Si no hay carpeta configurada,
    /// muestra NSSavePanel para elegir destino.
    func saveTranslatedFile(name: String, content: String) {
        if let folder = outputFolderURL {
            folder.startAccessingSecurityScopedResource()
            defer { folder.stopAccessingSecurityScopedResource() }
            let dest = folder.appendingPathComponent(name)
            try? content.write(to: dest, atomically: true, encoding: .utf8)
            NSWorkspace.shared.activateFileViewerSelecting([dest])
        } else {
            // Fallback: NSSavePanel
            let panel = NSSavePanel()
            panel.nameFieldStringValue = name
            panel.allowedContentTypes = [.init(filenameExtension: "md") ?? .text]
            guard panel.runModal() == .OK, let url = panel.url else { return }
            try? content.write(to: url, atomically: true, encoding: .utf8)
            NSWorkspace.shared.activateFileViewerSelecting([url])
        }
    }

    private func saveBookmark(for url: URL) {
        let data = try? url.bookmarkData(
            options: .withSecurityScope,
            includingResourceValuesForKeys: nil,
            relativeTo: nil
        )
        UserDefaults.standard.set(data, forKey: bookmarkKey)
    }
}
```

> **Security-Scoped Bookmarks** persisten el acceso a la carpeta entre reinicios sin pedir permiso cada vez. Funcionan sin App Sandbox (aunque son más relevantes con él).

### 2. Opción "Carpeta de salida…" en `SettingsView`

Añadir una sección en `SettingsView.swift`:

```swift
Section {
    HStack {
        if let folder = OutputManager.shared.outputFolderURL {
            Text(folder.lastPathComponent)
                .foregroundStyle(.secondary)
                .lineLimit(1)
                .truncationMode(.middle)
        } else {
            Text("Sin configurar (se pedirá al guardar)").foregroundStyle(.secondary)
        }
        Spacer()
        Button("Cambiar…") { OutputManager.shared.chooseFolderWithPanel() }
            .buttonStyle(.bordered)
    }
} header: {
    Text("Carpeta de archivos traducidos")
        .font(.subheadline).foregroundStyle(.secondary)
}
```

### 3. Puente JS→Swift para guardar y abrir el archivo

Cuando la traducción de archivo termina, el frontend JS tiene el resultado en el textarea de output. Usar un segundo `WKScriptMessageHandler` (`"saveTranslatedFile"`) para pasar el contenido a Swift:

```javascript
// En static/js/app.js, en el handler de éxito de /api/translate/file:
if (window.webkit?.messageHandlers?.saveTranslatedFile) {
    window.webkit.messageHandlers.saveTranslatedFile.postMessage({
        filename: translatedFilename,  // e.g. "documento_es.md"
        content:  translatedContent
    });
}
```

```swift
// En WebView.Coordinator.userContentController:
if message.name == "saveTranslatedFile",
   let body = message.body as? [String: String],
   let name = body["filename"], let content = body["content"] {
    OutputManager.shared.saveTranslatedFile(name: name, content: content)
}
```

Registrar el handler en `config.userContentController.add(context.coordinator, name: "saveTranslatedFile")`.

### 4. Menú "Carpeta de salida…" en `Commands.swift`

```swift
CommandGroup(after: .newItem) {
    Button("Carpeta de salida…") {
        OutputManager.shared.chooseFolderWithPanel()
    }
}
```

- [ ] Menú File > "Carpeta de salida…" accesible.
