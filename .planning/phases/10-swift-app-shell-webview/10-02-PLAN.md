# Plan 10-02 — Ventana nativa: menús, toolbar y UX macOS
## Phase: 10 — Swift App Shell & WebView

**Estado:** PLANIFICADO  
**Objetivo:** Integrar la app con las convenciones macOS: barra de menús con acciones útiles (abrir archivo, recargar), persistencia del tamaño de ventana, icono de app y About panel.

---

## Contexto

- La WKWebView (Plan 10-01) estará activa. La interfaz FastAPI ya tiene su propia UI para abrir archivos — el menú nativo la complementa para arrastra-y-suelta y acceso desde el Dock.
- `serverManager.serverPort` sigue siendo la pieza de conexión entre app y backend.
- `NSOpenPanel` es la forma estándar en macOS de pedir un archivo al usuario (sin necesitar entitlements especiales en desarrollo sin App Sandbox).

---

## Tareas

### 1. `Commands.swift` — Acciones nativas del menú

Añadir comandos SwiftUI para integrar con la barra de menús macOS:

```swift
// Commands.swift
import SwiftUI
import AppKit

struct AppCommands: Commands {
    let serverPort: Int

    var body: some Commands {
        // Reemplaza parte de CommandGroup(.newItem)
        CommandGroup(replacing: .newItem) {
            Button("Abrir archivo Markdown…") {
                openMarkdownFile(port: serverPort)
            }
            .keyboardShortcut("o")
        }

        CommandGroup(after: .windowArrangement) {
            Button("Recargar interfaz") {
                NotificationCenter.default.post(name: .reloadWebView, object: nil)
            }
            .keyboardShortcut("r")
        }
    }
}

extension Notification.Name {
    static let reloadWebView = Notification.Name("reloadWebView")
}

private func openMarkdownFile(port: Int) {
    let panel = NSOpenPanel()
    panel.allowedContentTypes = [.init(filenameExtension: "md")!]
    panel.allowsMultipleSelection = false
    panel.canChooseDirectories = false
    guard panel.runModal() == .OK, let url = panel.url else { return }
    // Leer el archivo y enviarlo al backend vía POST /api/translate/file (multipart)
    // o cargarlo en la WebView via JS (document.dispatchEvent con el contenido)
    // → Implementación en WebView.swift (método público callJavaScript)
    guard let content = try? String(contentsOf: url, encoding: .utf8) else { return }
    NotificationCenter.default.post(name: .openMarkdownContent, object: content)
}
```

> La WebView escuchará `Notification.reloadWebView` y `Notification.openMarkdownContent` vía `onReceive` en la vista.

### 3. Persistencia del tamaño de ventana con `UserDefaults`

`WindowGroup` persiste automáticamente el tamaño si la app tiene un identificador de scene estable. Añadir:

```swift
WindowGroup(id: "main") { ... }
```

macOS guarda el frame de la ventana en `UserDefaults` con la key derivada del bundle ID + scene ID. No requiere código adicional en SwiftUI si se usa `WindowGroup(id:)`.

Verificar en `~/Library/Preferences/com.edefrutos.md-translator.plist` tras resize + relanzar.

### 4. Icono de la app

- Crear `AppIcon.appiconset` en `Assets.xcassets` con los tamaños requeridos por macOS: 16, 32, 64, 128, 256, 512, 1024 (× 1× y 2× donde aplica).
- Herramienta sugerida: `makeicns` (Homebrew) o `iconutil` desde un PNG base de 1024×1024.
- El icono debe reflejar el concepto: documento Markdown + flecha de traducción.

```bash
# Generar AppIcon.icns desde un PNG 1024×1024
mkdir AppIcon.iconset
sips -z 16 16   icon_1024.png --out AppIcon.iconset/icon_16x16.png
# ... (todos los tamaños) ...
iconutil -c icns AppIcon.iconset
```

> **Nota:** Para Phase 10 es suficiente con un placeholder de color sólido. El icono definitivo puede hacerse en Phase 12 (distribución).

### 5. About panel personalizado

SwiftUI no ofrece `NSApp.orderFrontStandardAboutPanel` directamente. Usar `NSApplicationDelegateAdaptor`:

```swift
// En AppDelegate.swift, añadir:
func showAboutPanel() {
    NSApp.orderFrontStandardAboutPanel(options: [
        .applicationName: "MD Translator",
        .applicationVersion: "3.0",
        .credits: NSAttributedString(string: "MarkDown Auto Translator v3.0\nFastAPI + uvicorn embebido")
    ])
}
```

Y en `AppCommands`:
```swift
CommandGroup(replacing: .appInfo) {
    Button("About MD Translator") {
        NSApp.orderFrontStandardAboutPanel(nil)
    }
}
```
