# Requirements: MarkDown Auto Translator v3.1 — Native macOS Polish

## Overview

Milestone v3.1 convierte la app macOS de v3.0 en una app de primera clase: integración con Finder/Dock/Services, operación por teclado, y objetivos de rendimiento y calidad. Sin cambios en el backend Python (`src/`). Phase 16 (Distribution Upgrade) queda fuera de este milestone — bloqueada por Apple Developer Program.

**Stack:** Swift 6.3.2 / Xcode 26.5 / macOS 14+ · CPython 3.11.15 embebido · Sparkle 2.9.2

---

## v3.1 Requirements

### Phase 13 — Native macOS Integration (shipped 2026-06-10)

- [x] **DOCK-01**: Arrastrar `.md` al icono del Dock abre la ventana y carga el editor o lanza lote según número de archivos
- [x] **DOCK-02**: `NSApplication.dockTile` muestra progreso durante traducciones largas (`DockProgressManager.swift`)
- [x] **RECENT-01**: Menú `File > Open Recent` con los últimos 10 archivos
- [x] **DROP-01**: Drag & drop de `.md` sobre la ventana principal inyecta el contenido en el editor
- [x] **SERVICES-01**: Entrada "Traducir con MDTranslator" en el menú Services del sistema (`ServiceHandler.swift`, NSServices en Info.plist; fix de registro en e4860e5)

### Phase 14 — Keyboard & Workflow (shipped 2026-06-11)

- [x] **HOTKEY-01**: Atajo global configurable (por defecto `⌥⇧T`) — `GlobalHotkeyManager.swift`
- [x] **HOTKEY-02**: `⌘↩` lanza la traducción desde el editor
- [x] **HOTKEY-03**: `⌘⇧C` copia el resultado traducido al portapapeles
- [x] **ESTIMATE-01**: Estimación de tokens/coste en tiempo real vía `/api/translate/estimate` (debounced)
- [x] **UNDO-01**: `⌘Z` / `⌘⇧Z` funcionan en el textarea del editor dentro de WKWebView

### Phase 15 — Performance & Quality (shipped 2026-06-11)

- [x] **PERF-01**: `docs/performance.md` con métricas baseline v3.0 y metodología de medición
- [x] **PERF-02**: `python-bundle/` reducido a 116 MB (< 120 MB objetivo; baseline ~200 MB)
- [x] **PERF-03**: Arranque en frío < 5 s — **medido 2026-06-12: 1,46 s** (mediana de 3 runs, Mac Studio M2, `scripts/measure-cold-start.sh`); registrado en `docs/performance.md`
- [x] **CRASH-01**: Crash Reporter de Sparkle integrado (`CrashReporterManager.swift`)
- [x] **TEST-01**: `make smoke-test` — health check + `/api/translate/estimate` sin API key real
