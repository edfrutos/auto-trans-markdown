--- INICIO SECCIÓN REEMPLAZADA ---
## Instalación

### Con uv (recomendado — entorno exactamente reproducible)

Instala [uv](https://docs.astral.sh/uv/getting-started/installation/) si no lo tienes, luego:

```bash
cd auto-trans-markdown
uv sync                        # crea .venv e instala versiones exactas del lockfile
source .venv/bin/activate      # Windows: .venv\Scripts\activate
cp .env.example .env           # añade tu API key
```

Para incluir dependencias opcionales:

```bash
uv sync --extra test           # añade pytest y herramientas de test
uv sync --extra pdf            # añade WeasyPrint para export PDF
uv sync --all-extras           # instala todos los extras
```

### Con pip (alternativa sin cambio de toolchain)

```bash
cd auto-trans-markdown
python3 -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
pip install -e .               # instala el comando md-translate en .venv/bin/
cp .env.example .env           # añade tu API key
```

### Mantener el lockfile actualizado

```bash
# Añadir una nueva dependencia (actualiza pyproject.toml y uv.lock):
uv add <paquete>

# Actualizar un paquete específico manteniendo el resto bloqueado:
uv lock --upgrade-package <paquete>

# Actualizar todo a las últimas versiones compatibles:
uv lock --upgrade

# Regenerar requirements.txt tras cualquier cambio de lockfile:
uv export --format requirements-txt --no-hashes > requirements.txt
```
--- FIN SECCIÓN REEMPLAZADA ---
